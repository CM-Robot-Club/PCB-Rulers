# 说明
本项目源自[此项目](https://github.com/jbtronics/PCBruler)
* 继承了原项目中实用的功能，添加了个人的封装
* 丝印中添加了对原作者作品的引用说明
* 采用Altium Designer设计，并提供了此版本的原始加工文件
* 对于在嘉立创(JLC PCB)加工的需求，添加了"JLCJLCJLCJLC"丝印，可自定义客户编号位置
